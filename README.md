# Angular ES6 Template
A simple template to work with ES6 (ES2015) in AngularJS 1.x.x

## Instalation

```
npm install
npm install webpack -g
npm install http-server -g
```
## Build

```
webpack --watch
```
## Run server

```
http-server -p 3001
```

## Visualization

Just open [http://localhost:3001/](http://localhost:3001/) and you are good to go.
